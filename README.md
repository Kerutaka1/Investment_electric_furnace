# Investment_electric_furnace

An electric furnace capable of mass-producing basic materials in bulk.<br>
While the recipe requires a substantial quantity of materials and takes considerable time to complete, once constructed it boasts overwhelming production capacity, enabling the rapid mass production of basic materials.<br><br>
基本素材を一括大量生産できる電気炉。<br>
レシピに必要な素材数は非常に多く、完成まで時間がかかりますが、建設後は圧倒的な生産力で基本素材を一気に大量生産します。
